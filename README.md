# OIBSIP_WEB_TASK-3
Oasis Infobyte Internship

Domain: Web Development

Task 3 – Temperature Converter

Objective: To create a temperature converter using HTML, CSS and JavaScript.

Tools Used: HTML CSS JavaScript

Description: This project is a Temperature Converter web application developed as part of the Oasis Infobyte Internship. The application converts temperature between Celsius, Fahrenheit and Kelvin. JavaScript is used for conversion logic and CSS for styling.

Features: Celsius to Fahrenheit Fahrenheit to Celsius Kelvin conversion Simple UI

Result: Temperature Converter project completed successfully.
